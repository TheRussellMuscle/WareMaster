# WareMaster

A free, open-source, cross-platform companion app for the Japanese TTRPG **Wares Blade**.

WareMaster tracks characters, combat, and downtime activities (such as healing-over-time from wounds) so players and the Wares Maker can focus on the story instead of bookkeeping.

## Status

**Foundation phase.** No application code yet — this commit establishes:

- Stack decisions ([STACK.md](./STACK.md))
- Structured documentation of the Wares Blade rules under [`docs/`](./docs/README.md), used as the reference for everything the app will eventually do.
