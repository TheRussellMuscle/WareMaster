# 15 — Units and Glossary

> Source: Wares Blade Playkit, Index (p. 74)

## Units of measurement

| Unit | Approximate equivalent |
|---|---|
| **Etch** | 2 hours. (12 etches = 1 day.) |
| **Li** | ~4 km |
| **Liet** | ~4 m |
| **Litte** | ~4 cm |
| **Gro** | ~1.2 tons |
| **Gren** | ~1.2 kg |
| **Gran** | ~1.2 g |
| **Garom** | ~0.9 liters |
| **Golda** | Currency. 1 silver = 1 golda; 1 gold = 100 silver = 100,000 copper. |

## Time units used in play

| Unit | Real-time | Notes |
|---|---|---|
| **Segment (S)** | ~2 sec | Used during combat. |
| **Round (R)** | ~30 sec (15 S) | Rarely used; certain Rolls cost a Round. |
| **Turn (T)** | ~5 min (10 R) | WMs often use this for short timeframes (e.g. Technique durations). |
| **Hour (H)** | 1 half-etch (~1 hour) | Rarely used. |

## Player-facing acronyms

| Code | Meaning |
|---|---|
| **PC / NPC** | Player Character / Non-Player Character |
| **WM** | Wares Maker (game master) |
| **PP** | Proficiency Points |
| **IN** | Initiative Number |
| **DN** | Defense Number |
| **BN** | Battle Number |
| **D5 / D10** | Five-sided / ten-sided die |
| **G** | Golda (currency) |

## Beastiary stat-block fields

(See [`data/beastiary.yaml`](../data/beastiary.yaml).)

| Field | Meaning |
|---|---|
| **Rank** | A (deadliest) → E (easiest). |
| **CHA Modifier / Reaction** | Modifier added to CHA Rolls vs the monster + how it reacts on a failed Roll. |
| **Movement Speed [Sprint]** | Liets per Segment (sprint in brackets). |
| **Encounter Rate [Number Encountered]** | Likelihood within its habitat + how many appear together. |
| **`[bracketed values]`** | Stats vs Ryude — characters use the unbracketed values. |
