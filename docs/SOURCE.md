# Source

All game content in this repository is paraphrased and structured from the **Wares Blade Playkit** — an 82-page sample of the rulebook for the Japanese TTRPG *Wares Blade* (currently being translated into English).

This is a community companion tool, not affiliated with or endorsed by:
- The original Japanese publisher
- The English translation team

## Scope of the Playkit

The Playkit is a **sample**, not the full rules. As a result, this documentation captures only what the Playkit publishes:

- 4 character classes (Warrior, Word-Caster, Spiritualist, Tradesfolk)
- 6 ability scores, ~25 named skills
- A simplified character-creation walkthrough
- Action Rolls, combat, damage, advancement
- 8 Word-Casting Gates with 1–3 sample techniques each (Levels 1–2 only)
- Numetic Arts and Invocations (Levels 1–2 only)
- Western Ahan setting overview
- Ryude operation rules + 3 sample Ryude
- A 5-monster starter beastiary
- The character sheet layout

When the full English rulebook is released, this documentation will be expanded to match.

## Editing rules

If you spot a transcription error against the PDF, please open an issue with:
- The page number
- The text in the doc
- The text in the PDF

If a future Playkit revision changes mechanics, update both the relevant rules `.md` file and any affected `.yaml` records, and bump the `source` field.
